# Logos

All logo variations, formats, and treatments live here.

## Subfolder guide

- **`Primary/`** — the main logo. Default version used 80% of the time.
- **`Secondary/`** — alternate logo(s) for specific contexts (stacked, horizontal, wordmark).
- **`Icon/`** — icon-only mark. Used for app icons, favicons, small-space applications.
- **`Monochrome/`** — one-color versions (black, white, single-color brand).
- **`Favicon-App-Icons/`** — web favicons (ICO, PNG 32/64/180), iOS app icons, Android adaptive icons.
- **`Approved/`** — the current, approved versions. If it's in here, it's safe to use.
- **`Working-Files/`** — source files and drafts (AI, PSD, FIG). For designers only.
- **`Archive/`** — old approved versions. Don't delete — archive.

## Formats to keep

For every logo variation, keep:

- **SVG** — primary digital format
- **PNG** — transparent, multiple sizes (favicon, social, hero)
- **EPS or AI** — vector source for print
- **PDF** — print-ready, embedded fonts

## Naming

Follow the convention in `00_Start-Here/Naming-Convention.md`:

```
brand-logo-[primary|secondary|icon|monochrome]-[rgb|cmyk|web]-v[n]-YYYY-MM.ext
```

## What NOT to put here

- Branded social posts (→ `08_briefs-and-templates/asset-blueprints/social-media/` for the master template, `09_execution/channel-shipped/social/` for shipped instances)
- Logo lockups with other brands (→ a `Co-branding/` subfolder of `Logos/`)
- Founder headshots (→ `Photography/Team/`)
